# docs/history — archived process records

These are the archived internal development-process records of the
autonomous build protocol (see `CLAUDE.md`'s "Otonom Yürütme Protokolü")
that produced cli-comrade v0.1.0 through v0.1.4:

- **`UYGULAMA_PLANI.md`** — the master phase-by-phase implementation
  plan the protocol was executed against.
- **`PROGRESS.md`** — the phase-tracking ledger, updated at the end of
  every phase.
- **`phases/FAZ-XX.md`** — one report per phase, written on completion.

Kept for provenance only — this is not user-facing documentation. For
current install/usage docs see `../INSTALL.md` and the repo `README.md`.
